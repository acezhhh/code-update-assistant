# 代码更新助手
适用对当前java项目的依赖项目进行git拉取和maven打包，可减少我们在开发过程对依赖项目进行更新打包的繁琐操作。

### 环境

需要安装python3 运行main.py启动。

### 展示

选择jdk、项目目录、maven setting文件(一般公司都会用私服需要指定) 

选择完后点下一步会保存配置方便下次使用
